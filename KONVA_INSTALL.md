# Konva Installation Required

The AdminProductCreation page now uses Konva for canvas interactions. Please install the required packages:

```bash
npm install react-konva@^18.2.10 konva
```

Or with yarn:

```bash
yarn add react-konva@^18.2.10 konva
```

**Important:** Use react-konva version 18.x for React 18 compatibility. Version 19.x requires React 19.

The canvas components require these packages to function properly.
